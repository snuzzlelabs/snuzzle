# Snuzzle Landing Page

Simple one-page landing site for Snuzzle.

## Contents

- Fullscreen looping MP4 background
- Minimal HTML page
- Deployable to Vercel

## Deployment

1. Push repository to GitHub
2. Import project into Vercel
3. Connect domain (e.g. snuzzle.ai)

## Project Structure

index.html – landing page  
video.mp4 – background video
